## About the Project

This project is a frontend recreation of the Firefox homepage created for learning and practice purposes.

The goal was to improve my understanding of modern responsive web design, layout techniques, and utility-first styling with Tailwind CSS while recreating a real-world landing page as closely as possible.

## Built With

* HTML5
* Tailwind CSS
* Flexbox
* CSS Grid

## Features

* Fully responsive design
* Mobile-first approach
* Fixed navigation bar
* Complex card and grid layouts
* Modern gradients and visual styling
* Hover interactions and transitions
* Responsive typography and spacing

## What I Learned

Through this project, I practiced:

* Building responsive layouts for mobile, tablet, and desktop screens
* Using Flexbox for alignment and component layouts
* Using CSS Grid for complex section structures
* Working with Tailwind CSS utility classes efficiently
* Managing spacing, sizing, and typography across breakpoints
* Recreating a production-style landing page from an existing design
* Deploying a static website using GitHub Pages
